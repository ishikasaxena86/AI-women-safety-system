# AI Women Safety System

## 👩‍💻 Team Name
SheInnovates

## 🚨 Problem Statement
Women face safety issues in public places, especially at night. There is no real-time detection system and emergency response is often delayed.

## 💡 Solution
An AI-powered safety system that detects danger, sends instant alerts with live location, and provides quick emergency support.

## ⚙️ Tech Stack
- AI / Machine Learning
- Python
- GPS / Location Tracking
- Mobile Integration

## 🔥 Key Features
- Real-time danger detection
- Instant alert system
- Live location sharing
- Emergency response support
